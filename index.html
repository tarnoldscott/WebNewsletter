Under the hood
Platform & eng
⚛️
React 18 + Next.js 14 finally landed

Two upgrades that were a long time coming — and a meaningful unlock for everything we want to build next. Sentry errors dropped ~88% week over week when they landed. AI tooling made it possible to ship them faster than expected.

🔥
Firewall tuning cut compute by ~98%

A quiet but massive win: request middleware updates cut compute by ~98% on an average page load. Also added smarter traffic rules and reduced on-call alerts. The kind of thing nobody notices when it works — which is the point.

🔗
4,100 internal links being cleaned up

~4,100 internal links were resolving as redirects or errors. Remediation is underway. Unglamorous? Yes. Does it compound over time? Absolutely.

🛠️
CMS runbook is now a slash command

The CMS component checklist is now invokable as a slash command in Cursor. Fewer manual errors, faster build cycles. Small lift, real impact.

Learnings & experiments
4 takeaways
📐
Shorter pages convert better for paid traffic

An experiment sent 50% of paid traffic to a shortened Auth product page and produced a statistically significant conversion lift — enough to make it the default for that segment. The CTA-first, cut-the-fat approach works when the audience already has intent.

🧭
The right IA is still an open question — so it's being tested

A site-wide nav experiment is live with three variants across desktop and mobile, split via LaunchDarkly with results flowing into Databricks. The core question: how should plaid.com be structured to tell a product + solution + industry story without losing people? Real data incoming.

🗺️
Use case pages are now reusable patterns, not one-offs

Payments and Financial Management established a repeatable structure: context → value → how it works → proof. Written for the practitioner, not the API. A solid reference for the next wave of use case pages.

🤖
AI copy needs scaffolding, not buzzwords

The Intelligent Finance launch set a tone for AI writing at Plaid: more explanation, clearer hierarchy, less "AI-powered everything." Anchoring to the infrastructure layer — data + models + trust — gave the copy something concrete to stand on.
