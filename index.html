    <div class="learning">

      <div class="learning-icon">📐</div>

      <div class="learning-text">

        <strong>Shorter pages convert better for paid traffic</strong>

        <p>An experiment sent 50% of paid traffic to a shortened <a href="https://plaid.com/products/auth/">Auth product page</a> and produced a statistically significant conversion lift — enough to make it the default for that segment. The CTA-first, cut-the-fat approach works when the audience already has intent.</p>

      </div>

    </div>

    <div class="learning">

      <div class="learning-icon">🧭</div>

      <div class="learning-text">

        <strong>The right IA is still an open question — so it's being tested</strong>

        <p>A site-wide nav experiment is live with three variants across desktop and mobile, split via LaunchDarkly with results flowing into Databricks. The core question: how should plaid.com be structured to tell a product + solution + industry story without losing people? Real data incoming.</p>

      </div>

    </div>

    <div class="learning">

      <div class="learning-icon">🗺️</div>

      <div class="learning-text">

        <strong>Use case pages are now reusable patterns, not one-offs</strong>

        <p>Payments and Financial Management established a repeatable structure: context → value → how it works → proof. Written for the practitioner, not the API. A solid reference for the next wave of use case pages.</p>

      </div>

    </div>

    <div class="learning">

      <div class="learning-icon">🤖</div>

      <div class="learning-text">

        <strong>AI copy needs scaffolding, not buzzwords</strong>

        <p>The <a href="https://plaid.com/intelligent-finance/">Intelligent Finance launch</a> set a tone for AI writing at Plaid: more explanation, clearer hierarchy, less "AI-powered everything." Anchoring to the infrastructure layer — data + models + trust — gave the copy something concrete to stand on.</p>

    const w = canvas.offsetWidth, h = canvas.offsetHeight;

    ctx.clearRect(0, 0, w, h);


    const waves = [

      { amp: 28, freq: 0.012, speed: 0.018, color: 'rgba(62,207,207,0.35)', offset: 0 },

      { amp: 20, freq: 0.016, speed: 0.025, color: 'rgba(42,184,196,0.25)', offset: 1.2 },

      { amp: 18, freq: 0.009, speed: 0.012, color: 'rgba(120,100,220,0.2)', offset: 2.5 },

      { amp: 14, freq: 0.02,  speed: 0.03,  color: 'rgba(62,207,180,0.15)', offset: 0.7 },

    ];


    const lineCount = 18;


    waves.forEach(wave => {

      for (let i = 0; i < lineCount; i++) {

        const yBase = (h / (lineCount - 1)) * i;

        const spread = (i - lineCount / 2) / (lineCount / 2);

        ctx.beginPath();

        ctx.strokeStyle = wave.color;

        ctx.lineWidth = 0.8;

        for (let x = 0; x <= w; x += 3) {

          const y = yBase

            + Math.sin(x * wave.freq + t * wave.speed + wave.offset + spread * 1.5) * wave.amp * (1 - Math.abs(spread) * 0.4)

            + Math.sin(x * wave.freq * 0.5 + t * wave.speed * 0.7) * wave.amp * 0.3;

          x === 0 ? ctx.moveTo(x, y) : ctx.lineTo(x, y);

        }

        ctx.stroke();

      }

    });


    t++;

    requestAnimationFrame(draw);

  }


  resize();

  window.addEventListener('resize', () => { resize(); });

  draw();
